UCodeclose

A video streaming platform compiling the best coding tutorials organized in the form of channels and their respective topics.

A user must register and/or sign in, in order to access the content found within each channel. The channels contain a navbar displaying a list of technologies across the screen from which the user can select in order to display its video tutorials.

The application was built using React, React Router, Node, Express, MySQL, Sequelize, Materialize and Passport Authentication

Farmers' Marketsclose

An application built to locate nearby farmers' markets based on a user's zip code. Each result displays the market's name, address, map of location, and list of items on sale.
